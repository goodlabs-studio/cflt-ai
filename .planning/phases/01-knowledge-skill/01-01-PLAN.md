  <action>
    Extend tools/wiki-lint.py with the decay rule implementation:

    1. Add constant at module level (after imports, before get_wiki_root):
       `DECAY_DAYS = 90`

    2. Add two new functions after `parse_frontmatter()`:

       `check_decay(fm: dict, last_validated_fallback_field: str = "last_updated") -> bool`:
       - Return False if fm.get("confidence") != "high"
       - Get lv_raw = fm.get("last_validated") or fm.get(last_validated_fallback_field)
       - Return False if not lv_raw
       - Parse lv_date = datetime.strptime(str(lv_raw), "%Y-%m-%d"), return False on ValueError
       - Return lv_date < (datetime.now() - timedelta(days=DECAY_DAYS))

       `apply_decay_fix(content: str) -> tuple`:
       - Return (False, content) if not content.startswith("---")
       - Find end_idx = content.find("---", 3); return (False, content) if -1
       - Split: front = content[:end_idx + 3], body = content[end_idx + 3:]
       - Use re.sub(r'confidence:\s*high', 'confidence: medium', front, count=1)
       - Return (True, updated_front + body) if changed, else (False, content)

    3. In `lint_wiki()` function signature, add `fix: bool = False` parameter.
       In the `findings` dict, add `"decayed": []` entry.
       In the per-article loop, after the existing "Stale (last_updated > 90 days ago)" block, add:
       ```
       # Decay rule (WIKI-03/04): confidence:high with stale last_validated
       if check_decay(fm):
           findings["decayed"].append(rel)
           if fix:
               changed, new_content = apply_decay_fix(content)
               if changed:
                   md.write_text(new_content)
       ```

    4. In `main()`:
       - Add argparse argument: `parser.add_argument("--fix", action="store_true", help="Auto-fix: demote stale confidence:high to medium")`
       - Add logic: `if args.fix: args.full = True` (--fix implies --full)
       - Update lint_wiki call: `findings = lint_wiki(root, full=args.full, fix=args.fix)`

    5. In the `labels` dict in main(), add: `"decayed": "Decayed articles (confidence demoted)"`
